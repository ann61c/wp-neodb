# WP-NeoDB 5.0

可以以豆列的方式展示电影、图书、音乐，游戏，舞台剧。
可以在文章中插入NeoDB, 豆瓣，TMDB 链接，自动获取数据。

数据本地化，解决豆瓣接口失效及图片防盗链。

latest version `5.0.3`

## 快速配置

[帮助文档](https://blog.lyc.sh/wp-neodb)
